Welcome to Discord Bot Maker v3.0.0!

New Features

This update aims to provide all the new features for Discord bots! This includes slash commands, context menu commands, buttons, select menus, and more!

However, since this update is so big and so many things were changed, there may be a couple problems when trying to begin. Please read through this information before proceeding!


Existing Projects

To safetly ensure there are no problems, your last project was not opened automatically.

It may contain mods that are not compatible with this update. There may be other issues that occur, so be sure to make a copy of your projects before opening in this new update.

It's recommended you create a new project at File > Create New Project!
However, you can go to File > Open Existing Project to open an old project.

If you would like to use the previous version of Discord Bot Maker, visit Steam and go to Properties... > BETAS and select old_version_v1.6.


Existing Themes

Themes from the previous version of Discord Bot Maker are also no longer compatible. As a result, your current theme has been set to default.

Old themes can still be selected again from the Theme menubar.


Porting Old Project

If you would like to use an old project, please follow the following steps after opening it:

• Use Project > Module Manager and clear the version text on discord.js. It should say ^13.2.0. Click "Save Modules" and the project should update.

• Use File > Actions/Events/Extensions > Update Default Actions/Events/Extesions to install the new actions into your project.

• Use File > Music Capabiltiies > Update Music Libraries to install the new music libraries.

• Use the invite link in SETTINGS > Bot Settings to re-invite the bot into the server you test it with. You do NOT need to kick the bot. This just gives it the permissions to setup Slash Commands in the server.

If your bot is in a lot of servers, go to Slash Command Options > Slash Command Creation Preference and choose an option that enables global commands. Global commands do not require re-invite, but take a long time to update (about 1 hour), so they are not very good for testing.
